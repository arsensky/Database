# Database Course (COM-213)
This repository contains materials for the **Database** course (COM-213), Spring 2026.

## Course Overview
The course focuses on database fundamentals, SQL, and practical database usage.  
All required labs and additional study materials are organized into separate folders/repositories.

## Labs
- All **Labs (1–17)** are completed.
- Labs are located in the **`Labs/`** directory.
- **Lab 1** and **Lab 10** are short and theoretical, so they are written in **`.md`** format.
- All other labs are implemented in **`.sql`** format.
- The **`lab10_images/`** folder contains screenshots related to **Lab 10**.

## Cheat Sheet
- The **Cheat Sheet** was created by me based on lab materials.
- It is intended to help quickly review:
  - Important concepts  
  - SQL commands  
  - Key details from the course
- Located in the **`Cheat sheet/`** directory.
